# Football League API

RESTful Web API for managing football league standings.

## Technologies
- ASP.NET Core (.NET 5)
- Entity Framework Core
- SQL Server
- MediatR (Mediator pattern)
- Swagger
- xUnit (Unit Tests)

## Features
- CRUD for Teams
- CRUD for Matches
- Automatic standings calculation
- Validation for existing teams
- Scoring system (Win: 3, Draw: 1, Loss: 0)

## Run the project
1. Update connection string in appsettings.json
2. Run the project
3. Swagger available at /swagger
